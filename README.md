# Homebrew Tap for claudectx

This is the official Homebrew tap for [claudectx](https://github.com/foxj77/claudectx).

## Installation

```bash
brew install foxj77/tap/claudectx
```

## What is claudectx?

Fast, safe profile switching for Claude Code. Switch between Claude Code configurations in seconds.

For more information, visit the [main repository](https://github.com/foxj77/claudectx).

## Formulas

- **claudectx** - Fast, safe profile switching for Claude Code

## Updating

Formulas in this tap are automatically updated by [GoReleaser](https://goreleaser.com/) when new releases are created.
